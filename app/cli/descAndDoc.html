Description and aplication documentation


About this aplication
Welcome to the eCommerce Order Management System – a comprehensive web application designed to streamline the process of creating, viewing, and managing online orders. This application provides an intuitive interface for handling customer orders, tracking inventory, and managing product sales with automatic IVA (VAT) calculations. Built with modern web technologies, this system combines a user-friendly client-side interface (HTML, CSS, and JavaScript) with a robust server-side backend (PHP) to ensure reliable data storage and retrieval. All order information is securely stored in a binary database file, ensuring data persistence and quick access to order history. Key Features: Create Orders: Add new customer orders with multiple products and automatic price calculations View All Orders: Browse complete order history with detailed statistics and revenue tracking Search Orders: Quickly find specific orders by their unique ID IVA Calculation: Automatic 21% VAT calculation on all orders Real-time Updates: Instant feedback on order creation and search operations
How to use the operation menu

When you access the application, you'll be presented with three main operations. Here's how to use each one:
1. Create Order Use this option to register a new customer order in the system. Steps: Select the "Create order" radio button Click "Select" to proceed Fill in all customer information: Command ID (unique order identifier) Customer name, last names Email address and phone number Delivery address Select products from the list by checking the boxes Adjust quantities for each selected product (minimum 1) Click "Send" to process the order The system will display the total price with IVA (21%) Available Products: Bicycle carrier: 80€ Snow chains: 60€ GPS: 120€ Spare tire: 130€
2. View All Orders Access a complete overview of all orders stored in the system. What you'll see: Statistics Dashboard: Total orders, revenue, IVA collected, and average order value Detailed Table: All orders with customer information, products, prices, and dates Orders are sorted by date (newest first) Each row displays: Order ID, customer details, products purchased, quantities, pricing breakdown, and creation date
3. Search Order by ID Quickly locate and view details of a specific order.
Steps:
Select the "Search order by ID" radio button Click "Select" to proceed Enter the Command ID in the search field Click "Search Order" View complete order details including: Customer information Order date and time Complete product list with quantities Price breakdown (without IVA, IVA amount, total with IVA) Navigation Back to Menu: Returns you to the operation selection menu Back to Home: Takes you to the application's main homepage
